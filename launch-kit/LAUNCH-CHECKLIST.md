## Emergency Responses

### If HN post flops
- Don't panic. Try reposting with a different title.
- Focus on Reddit and Twitter instead.

### If someone criticizes
- Thank them for the feedback
- Ask clarifying questions
- Be humble and open
- Turn critics into contributors

### If it goes viral unexpectedly
- Pin a tweet/post with the GitHub link
- Prepare for increased GitHub issues
- Consider adding a CONTRIBUTING.md quickly
- Ride the wave! 🌊
