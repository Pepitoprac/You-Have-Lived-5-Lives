# Quick Start - Executive Strategist 5 Lives

## 30-Second Setup

1. **Copy files to your project:**
   ```bash
   cp -r executive-strategist/* /your/project/
   ```

2. **Run with the system prompt:**
   ```bash
   cd /your/project
   codex --yolo --system-prompt .agent/system-prompt.md \
     "Using an ExecPlan, develop a competitive strategy"
   ```

3. **Watch the magic** ✨

## Magic Words

Trigger the full 5 Lives experience:

| Trigger | Use Case |
|---------|----------|
| "Using an ExecPlan" | Activates planning protocol |
| "strategy" "competitive advantage" | Full strategic analysis |
| "business model" | Business model canvas, unit economics |
| "growth" "scaling" | Growth strategy with execution roadmap |
| "Blue Ocean" | Differentiation and value innovation |

## Pro Tips

1. **Enable Web Search!** Add to `~/.codex/config.toml`:
   ```toml
   [features]
   web_search = "live"
   ```
   This lets the Executive Strategist research current industry trends, competitor moves, and market dynamics!

2. **Provide industry context** for relevant frameworks
3. **Mention competitors** for positioning analysis
4. **Ask about trade-offs** (strategy always involves choices)
5. **Request specific frameworks** (Porter, BCG Matrix, Blue Ocean, etc.)
6. **Include time horizon** (tactical vs. strategic vs. transformative)

## Strategic Frameworks Available

- **BCG Matrix**: Portfolio prioritization
- **Ansoff Matrix**: Growth strategies
- **Business Model Canvas**: Nine building blocks
- **Value Chain Analysis**: Where value is created
- **Blue Ocean Strategy**: Value innovation
- **Core Competencies**: Sustainable advantage
- **OKRs**: Strategy execution

## Troubleshooting

**Output too generic?**
- Provide: Specific industry, competitors, current position
- Ask for: Specific framework application

**Need more execution detail?**
- Add: "with implementation roadmap" or "with organizational implications"

**Want competitive analysis?**
- Say: "analyze using Porter's Five Forces" or "competitive positioning map"
