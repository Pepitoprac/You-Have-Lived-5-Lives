# Quick Start - Financial Analyst 5 Lives

## 30-Second Setup

1. **Copy files to your project:**
   ```bash
   cp -r financial-analyst/* /your/project/
   ```

2. **Run with the system prompt:**
   ```bash
   cd /your/project
   codex --yolo --system-prompt .agent/system-prompt.md \
     "Using an ExecPlan, create an investment strategy"
   ```

3. **Watch the magic** ✨

**⚠️ Remember: This is educational, not personalized financial advice.**

## Magic Words

Trigger the full 5 Lives experience:

## Pro Tips

1. **Enable Web Search!** Add to `~/.codex/config.toml`:
   ```toml
   [features]
   web_search = "live"
   ```
   This lets the Financial Analyst research current market data, recent earnings, and economic indicators!

2. **Specify your risk tolerance** (conservative/moderate/aggressive)
3. **Mention your time horizon** (short/medium/long-term)
4. **Include your current portfolio** for context
5. **Ask about tax optimization** for taxable accounts
6. **Request behavioral coaching** if you struggle with emotional decisions

## Troubleshooting

**Need more specific advice?**
- Provide: Age, income, risk tolerance, time horizon, current assets

**Want trading strategies?**
- Specify: Timeframe (day/swing/position), asset class, risk tolerance

**Concerned about taxes?**
- Mention: Account types (taxable/401k/IRA), tax bracket, state
