# Quick Start - Cybersecurity Expert 5 Lives

## 30-Second Setup

1. **Copy files to your project:**
   ```bash
   cp -r cybersecurity-expert/* /your/project/
   ```

2. **Run with the system prompt:**
   ```bash
   cd /your/project
   codex --yolo --system-prompt .agent/system-prompt.md \
     "Using an ExecPlan, secure this application"
   ```

3. **Watch the magic** ✨

## Magic Words

Trigger the full 5 Lives experience:

| Trigger | Use Case |
|---------|----------|
| "Using an ExecPlan" | Activates planning protocol |
| "secure" "hardened" | Full security analysis |
| "zero trust" | Zero trust architecture |
| "compliance" | SOC2, GDPR, ISO27001 |

## Pro Tips

1. **Mention compliance needs** (SOC2, GDPR, HIPAA)
2. **Say "production"** for enterprise-grade security
3. **Include "zero trust"** for modern architecture
4. **Use "threat model"** for proactive security analysis
5. **Say "defense in depth"** for layered security

## Troubleshooting

**AI not doing threat modeling?**
- Use: "Include STRIDE threat modeling" or "threat analysis"

**Output lacks monitoring?**
- Add: "with SIEM monitoring" or "comprehensive logging"
