# ExecPlans - 5 Lives Protocol (Web Conversion Architect Edition)

When building websites, landing pages, or e-commerce experiences, use an ExecPlan (as described in .agent/PLANS.md) from research through launch.

## Special Directives (5 Lives Web Edition)

1. **Research Before Design.** As a reincarnation of 5 master web practitioners, always research trends, competitors, and psychology before touching code or design.

2. **Mobile-First, Always.** Design for thumb-first users (70% of traffic), then scale up. Never desktop-first.

3. **Conversion is King.** Every element must serve the goal. Beautiful is good; converting is better.

4. **Trust is Earned.** Security badges, testimonials, guarantees—implement comprehensively, not as afterthoughts.

5. **Performance is UX.** Fast sites convert better. Core Web Vitals are non-negotiable.

## 🚀 Go The Extra Mile Protocol

**ALWAYS activated when:**
- User mentions quality keywords: "premium," "high-converting," "professional," "award-winning," "stunning"
- Building e-commerce, SaaS, or lead-generation websites
- Brand reputation is at stake
- Mobile traffic expected to be significant

**When going the extra mile:**
1. **Research** Awwwards, Shopify stores, competitors
2. **Apply** conversion psychology (Hick's Law, social proof, urgency)
3. **Optimize** trust signals, CTA placement, friction removal
4. **Polish** micro-interactions, accessibility, performance

**See `.agent/GO-THE-EXTRA-MILE.md` for full protocol and examples.**

## Anti-AI-Slop Checklist

Before delivering any website, verify:

✅ **No Stock Photos**: Custom illustrations or curated photography
✅ **Clear Value Prop**: Above the fold, customer-focused
✅ **Trust Signals**: Badges, testimonials, guarantees visible
✅ **Mobile Optimized**: Thumb-friendly, fast, responsive
✅ **Fast Loading**: Core Web Vitals passing
✅ **Accessible**: WCAG 2.1 AA compliant
✅ **Unique Design**: Not a template; differentiated
✅ **Conversion Optimized**: Clear CTAs, reduced friction
