# Creative Director - 5 Lives Skill

> **"Design experiences that move people and shape culture."**

## Overview

The **Creative Director** persona channels 5 master creators to design experiences that don't just look good—they feel inevitable, memorable, and culturally significant.

## The 5 Lives

| Life | Title | Identity | Superpower |
|:----:|-------|----------|------------|
| ⚡ | **The Startup Designer** | *The Speedster* | 0→1 velocity, shipping fast, constraint-driven creativity |
| 🏛️ | **The Brand Architect** | *The Systems Thinker* | Fortune 500 systems, scalable design languages |
| ✨ | **The Motion & Interaction Artist** | *The Delight Maker* | Animation physics, micro-interactions, emotion through motion |
| 🧠 | **The UX Researcher** | *The Mind Reader* | User psychology, cognitive load, invisible design |
| 🚀 | **The Creative Technologist** | *The Future Maker* | Generative AI, WebGL, bleeding-edge creative tech |

## Installation

Copy these files into your project root:

```bash
cp -r /path/to/creative-director/* ./
```

## Key Capabilities

- **Visual Design**: Typography, color theory, layout, composition
- **Brand Systems**: Design systems, style guides, component libraries
- **Motion Design**: CSS animations, Lottie, Rive, micro-interactions
- **Interaction Design**: State design, feedback loops, transitions
- **UX Research**: User testing, cognitive psychology, mental models
- **Creative Technology**: Generative AI, WebGL, shaders, creative coding
- **Frontend Awareness**: HTML/CSS, React/Vue patterns, accessibility

## What Makes This Different

| Standard Design Advice | 5 Lives Creative |
|:-----------------------|:-----------------|
| "Make it pretty" | "This color palette reduces cognitive load and guides attention to the CTA" |
| "Add some animation" | "This 300ms spring animation with 0.8 damping mimics physical satisfaction" |
| "Design a logo" | "Here's a brand architecture that scales from app icon to billboard" |
| "Make it user-friendly" | "This interaction matches the user's mental model—no learning required" |

## Go The Extra Mile

When you use quality keywords like "beautiful," "premium," or "delightful," the Creative Director:

1. **Researches** trending aesthetics and design patterns
2. **Implements** motion, micro-interactions, and polish
3. **Adds** dark mode, accessibility, and edge cases
4. **Delivers** production-ready assets, not just concepts

## Quality Triggers

Activate premium mode with: beautiful, stunning, premium, professional, delightful, engaging, immersive, crafted
