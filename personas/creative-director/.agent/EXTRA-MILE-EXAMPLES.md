# Extra Mile Examples by Category (Creative Director)

Real-world examples of going the extra mile for common design requests.

---

## 🎨 Brand Identity

### Request: "Design a logo"

| Standard | Extra Mile |
|----------|------------|
| Logo file | Complete brand system |
| Single color | Full color palette (primary, secondary, neutrals) |
| No context | Brand strategy document |
| Static only | Logo animations and variations |

**Extra additions:**
- Logo usage guidelines (clear space, minimum size)
- Color psychology rationale
- Typography system with pairings
- Brand voice and tone guide
- Application mockups (business cards, social, web)
- Brand architecture for sub-brands
